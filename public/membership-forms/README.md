# Membership Forms Organization

This folder contains the membership application forms and related documents for BAA (Buying Agents Association).

## Folder Structure

```
membership-forms/
├── templates/
│   └── (Blank/template forms stored here)
└── submissions/
    └── (User-submitted forms stored here)
```

## Templates Folder (`/templates/`)

This folder contains the **blank membership application form templates** that users can download and fill out offline.

**Files:**
- `BAA-Membership-Form-Blank.pdf` - Standard blank form template

**Usage:**
- Users can download these as reference templates
- Forms downloaded from the website are auto-populated versions of these templates

## Submissions Folder (`/submissions/`)

This folder is designated for storing completed membership forms that users have uploaded.

**Organization:** Forms are organized for easy retrieval:
- By date (YYYY-MM-DD)
- By membership category
- Named pattern: `BAA-Form-[UserName]-[Date].pdf`

**Managing Files:**
- Regularly archive old submissions (60+ days)
- Review and process applications
- Keep records for compliance

## Workflow

### Step 1: User Downloads Form
- User enters details in the online form
- System generates personalized PDF with pre-filled user information
- PDF is automatically named: `BAA-Membership-Form-[UserName].pdf`

### Step 2: User Fills Form (Offline)
- User prints or fills the PDF on computer
- User gathers required documents:
  - Identity Proof (Aadhaar/PAN/Passport)
  - Business Registration Certificate
  - Bank Details for Payment
  - Company Profile/Brochure (if applicable)

### Step 3: User Uploads Form
- User selects completed form file
- System prepares for email submission
- Email client opens with pre-filled submission email
- User attaches the completed form and sends to info@baa.org.in (CC: gs@baa.org.in)

## PDF Generation

- **Generated by:** `/app/api/generate-pdf/route.ts`
- **Format:** jsPDF library generates professional PDFs
- **Pre-filled Fields:**
  - Name
  - Email
  - Phone
  - Company Name
- **Output:** Base64 encoded PDF with data URI for instant download

## File Handling

- Maximum file size for uploads: 10 MB
- Accepted formats: PDF, Word (.doc, .docx), Images (.jpg, .jpeg, .png)
- Files are NOT stored on server (using mailto: for submission)
- Users must email files directly to info@baa.org.in
